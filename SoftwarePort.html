IonEngine - Custom Unity-like Game Engine

IonEngine is an open-source game engine I designed in a single semester inspired by the Unity Game Engine. It features a custom editor built using the Dear ImGui library designed to feel familiar to Unity developers, a custom data-oriented particle system, a physics engine utilizing Box2D, scripting capabilities using Lua, and more.

Engine Stack

  • C++ - The backbone of both the runtime engine and the editor, delivering high performance and control.
  • LuaBridge - A lightweight library that binds C++ functions and classes to Lua. Implemented to make scripting seamless and intuitive.
  • SDL - Cross-platform library used to control windowing, input, audio, and rendering on Windows, macOS, and Linux.
  • Box2D - The physics engine that controls colliders, triggers, and rigidbody physics.
  • Dear ImGui - Immediate mode GUI used to build IonEngine's custom in-editor interface.
  • RapidJSON - All information regarding scenes, actors, and components is stored in JSON files that need to be read/written quickly.

I was the sole developer working on this engine with weekly check-ins on progress. Starting with a text adventure game, I rapidly evolved IonEngine to include 2D rendering and audio output using the SDL library, allow scripting through Lua and the LuaBridge library, integrate a full editor to provide ease of development, and utilize rigidbody physics to simulate movement.

I also implemented a custom data-oriented particle system that was fully written in C++ and accessible in Lua through the LuaBridge library. This allowed millions of particles to be rendered on screen and simulated with minimal performance loss.

ArcJet - 2D / 3D Game Engine

ArcJet is an open-source game engine that I'm currently developing. It is a fully data-oriented engine with support for both 2D and 3D graphics using OpenGL. I am also deepening my understanding of computer graphics as I implement OpenGL into the engine.

The finished engine will support 2D and 3D graphics, allow scripting through C#, Python, and Lua, and include a built-in custom editor for ease of development.
